## Development
To run a local NABI UI development server and view the application in your browser:

#### Step 1. Clone repo
`git clone https://github.com/iamvane/nabi_ui.git`

#### Step 2. Install dependecies
`npm install`

#### Step 3. Start server
`npm start`

## Tests
To run tests use `npm test`
